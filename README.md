# Valhalla Tattoo Studio Website

A professional tattoo studio website featuring artist portfolios, booking system, and comprehensive image galleries. Built for Valhalla Tattoo in Spring Hill, TN.

## 🌟 Features

### ✅ Complete Portfolio System
- **Individual Artist Portfolios** - Dedicated pages for each of 5 talented artists
- **Advanced Image Galleries** - Responsive CSS Grid with lazy loading
- **Lightbox/Modal Viewing** - Full-screen image experience with navigation
- **Smart Filtering System** - Filter by tattoo styles and tags
- **Booking Integration** - Seamless artist pre-selection and booking flow

### 🎨 Design & User Experience
- **Consistent Design System** - Cohesive color palette and typography
- **Responsive Design** - Perfect experience across all devices
- **Smooth Animations** - Professional transitions and hover effects
- **Accessibility Features** - ARIA labels, keyboard navigation, screen reader support

### 🚀 Technical Features
- **Performance Optimized** - Progressive image loading and lazy loading
- **Error Handling** - Comprehensive error boundaries and fallback states
- **SEO Optimized** - Proper meta tags and semantic HTML structure
- **Modern JavaScript** - ES6+ with modular architecture

## 📱 Responsive Breakpoints

- **Desktop Large** (1200px+) - 4-column portfolio grid
- **Desktop** (992px-1199px) - 3-column portfolio grid
- **Tablet** (768px-991px) - 2-column portfolio grid
- **Mobile** (below 768px) - 1-column portfolio grid
